Software Development - Index of Shakespeare's Complete Works

This program is designed to demonstrate a simple client/server program. A server program runs in the background, waiting for a search requests. When it receives a search request it returns the lines that contain the searched word.

The reason for such a architecture is to allow a simple program, with a MAP preloaded with data, to quickly provide results without incurring the overhead of a reloading the texts for each request.
